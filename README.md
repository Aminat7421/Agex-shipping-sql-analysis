# AgeX Shipping SQL Analysis

This repository contains the SQL analysis project for **AgeX Shipping**, based on the provided ER diagram and project report. It demonstrates the use of SQL to query, extract, and analyze key shipping data to understand operational metrics and trends.

---

## Repository Structure
```
agex-shipping-sql-analysis/
│── README.md
│── ERD/
│ └── your_erd_image.png
│── docs/
│ └── AgeX_Shipping_Report.pdf
│── sql/
└── agex_shipping_analysis.sql
```

---


- **ERD/** – Contains the Entity-Relationship Diagram for the AgeX Shipping database.  
- **docs/** – Contains the project report with detailed analysis and findings.  
- **sql/** – Contains the SQL script (`agex_shipping_analysis.sql`) used for querying and analyzing the shipping data.

---

## SQL Script Overview

The SQL script includes queries to:

- Retrieve and summarize shipping orders and transactions.
- Analyze shipment trends and performance metrics.
- Identify key operational insights based on the database schema.
